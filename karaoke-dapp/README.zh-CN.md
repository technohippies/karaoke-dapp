# Karaoke dApp

基于 Base Sepolia 的完全去中心化卡拉OK平台。无服务器，无中心化数据存储。

## 去中心化架构

这是一个**无服务器的去中心化应用**：
- 无后端服务器或中心化基础设施
- 用户通过区块链和IPFS控制自己的数据
- 作为非营利性公共产品运营

## 版权声明

本项目**无法也不会托管任何受版权保护的内容**：

- **歌词**：由用户直接从第三方API（LRCLIB）获取。无中间服务器。
- **图片**：在用户浏览器中直接从外部源（Genius API）加载。无代理或缓存。
- **MIDI文件**：仓库仅包含程序生成的演示文件。所有用户内容存储在去中心化网络（IPFS/Arweave）。
- **音频**：通过MIDI文件程序化生成，加密并跟踪版税分配。

## 版税合规

本平台完全遵守音乐许可法规：
- **MLC合规**：遵循机械许可集体的支付时间表和法规
- **版税跟踪**：所有使用情况在链上跟踪，确保版税分配透明
- **自动支付**：通过DNS验证向确认地址分配版税
- **权利人优先**：我们积极确保所有相关方获得应得的版税支付

没有中心化基础设施来接收下架请求 - 所有内容决策由个人用户做出。
