# Karaoke dApp Architecture V2 (Updated)

## System Overview
Decentralized karaoke learning platform combining real-time voice grading, spaced repetition (FSRS), and Duolingo-style exercises. Users own their learning data through encrypted Tableland tables.

**Voice Credit Model**: 1 voice credit = 1 karaoke line attempted OR 1 exercise attempt (e.g., "Say it back"). Simple, predictable pricing where users pay $1 USDC for 100 voice credits.

**Data Storage Strategy**: 
- **During Karaoke**: Store line results in IndexedDB with Lit PKP signatures for tamper-proofing
- **After Karaoke**: Write aggregated SRS data to user's Tableland tables (created on first save, not signup)
- **No Recall Network**: Simplified architecture without additional dependencies

#### Session Settlement Action (PKP Signing)
```javascript
const sessionSettlementAction = async () => {
  // Parameters: userId, sessionId, creditsUsed, pkpPublicKey
  
  // 1. Validate session data
  if (!userId || !sessionId || !creditsUsed) {
    throw new Error("Missing required parameters");
  }
  
  // 2. Create settlement message for smart contract
  const messageData = ethers.utils.solidityPack(
    ["address", "bytes32", "uint256"],
    [userId, sessionId, creditsUsed]
  );
  
  const messageHash = ethers.utils.keccak256(messageData);
  
  // 3. Sign with PKP
  const sigShare = await Lit.Actions.signEcdsa({
    toSign: ethers.utils.arrayify(messageHash),
    publicKey: pkpPublicKey,
    sigName: "settlement"
  });
  
  // 4. Return signed settlement
  Lit.Actions.setResponse({
    response: JSON.stringify({
      success: true,
      settlement: {
        userId,
        sessionId,
        creditsUsed,
        messageHash,
        timestamp: Date.now()
      },
      signature: sigShare.signature
    })
  });
};
```

async function completeKaraokeSession(sessionData) {
  // 1. Aggregate line results from IndexedDB
  const lineResults = await getSessionResults(sessionData.sessionId);
  const creditsUsed = lineResults.length;
  
  // 2. Get session signatures using wallet auth
  const sessionSigs = await litNodeClient.getSessionSigs({
    pkpPublicKey: PKP_V3_PUBLIC_KEY,
    authMethods: [walletAuthMethod],
    resourceAbilityRequests: [sessionSettlementResource]
  });
  
  // 3. Execute settlement Lit Action
  const result = await litNodeClient.executeJs({
    sessionSigs,
    ipfsId: SESSION_SETTLEMENT_CID,
    jsParams: {
      userId: user.address,
      sessionId: sessionData.sessionId,
      creditsUsed,
      pkpPublicKey: PKP_V3_PUBLIC_KEY
    }
  });
  
  // 4. Submit settlement to smart contract
  const settlement = JSON.parse(result.response);
  await karaokeStore.settleVoiceSession(
    settlement.settlement.userId,
    settlement.settlement.sessionId,
    settlement.settlement.creditsUsed,
    settlement.signature
  );
  
  // 5. Save SRS data to user tables
  await updateUserTables(lineResults, sessionData);
}

## 6. Security & Anti-Tampering

### IndexedDB Signature Verification
```typescript
// lib/session-storage.ts
export class SessionStorage {
  async storeLineResult(lineResult: LineResult, sessionId: string) {
    // Sign result with PKP for tamper-proofing
    const signature = await this.litClient.signData(
      JSON.stringify(lineResult),
      sessionId
    );
    
    const signedResult = {
      ...lineResult,
      pkpSignature: signature,
      timestamp: Date.now()
    };
    
    await this.idb.put('lineResults', signedResult);
  }
  
  async getSessionResults(sessionId: string): Promise<LineResult[]> {
    const results = await this.idb.getAll('lineResults', sessionId);
    
    // Verify all PKP signatures
    const verified = await Promise.all(
      results.map(result => this.verifyPKPSignature(result))
    );
    
    return verified.filter(result => result.valid);
  }
}
```
